Proven 8-Week Speed Plan (overview)

Follow a cyclical plan that mixes aerobic base, speed work, strength, and recovery.

  1. Build an aerobic base (Weeks 1–2)
    • Run easy 4–5x/week — keep most runs conversational
    • Include one longer run (gradually increase time)
    • 2x strength sessions (light, focused on single-leg strength)
  2. Add tempo & threshold work (Weeks 3–4)
    • 1 tempo run (20–30 min at 'comfortably hard')
    • 1 easy long run + recovery days
  3. Speed & intervals (Weeks 5–6)
    • Track intervals (e.g., 6×400m or 4×800m) — hard but controlled
    • Include strides (6×20–30s) after easy runs
  4. Sharpen + taper (Weeks 7–8)
    • Reduce total volume; maintain intensity
    • Race simulation or time trial

Key Technique & Drills

Practice these drills to improve form and efficiency:

  • Stride mechanics (focus on quick turnover)
    1. Drill 1: High knees — 3×30m
    2. Drill 2: Butt kicks — 3×30m
    3. Drill 3: A/B/C running drills — 2× each
  • Arm drive: keep elbows ~90°, hands relaxed
  • Cadence: aim for ~170–190 steps/min (individual variation applies)
